Sıkça Sorulan Sorular
Ücretsiz deneme nasıl çalışır?
Kayıt olduktan sonra 14 gün boyunca seçtiğiniz planın tüm özelliklerini ücretsiz kullanabilirsiniz. Deneme süresinde kredi kartı bilgisi istenmez. Süre sonunda otomatik ücretlendirme yapılmaz — planınızı aktif etmek için ödeme bilgilerinizi girmeniz gerekir.
Planımı istediğim zaman değiştirebilir miyim?
Evet, Starter'dan Pro'ya istediğiniz zaman yükseltme yapabilirsiniz. Fark orantılı olarak hesaplanır. Pro'dan Starter'a geçiş ise mevcut dönem sonunda uygulanır.
Hangi platformlar destekleniyor?
Şu anda Trendyol, Hepsiburada, Amazon TR ve N11 desteklenmektedir. Her platform için API entegrasyonu, komisyon oranları ve kategori verileri mevcuttur. Yeni platformlar sürekli eklenmektedir.
Verilerim güvende mi?
Tüm verileriniz Firebase altyapısı üzerinde şifrelenmiş olarak saklanır. API anahtarları KMS envelope encryption ile korunur ve plaintext olarak hiçbir yerde tutulmaz. KVKK uyumlu veri işleme politikamız mevcuttur.
API bağlantım ne kadar güvenli?
Pazaryeri API kimlik bilgileri Google Cloud KMS ile şifrelenir, yalnızca backend tarafından çözülebilir. Hiçbir credential frontend'e geri dönmez ve istek logları içinde görünmez. Bağlantıyı dilediğiniz zaman silebilirsiniz.
